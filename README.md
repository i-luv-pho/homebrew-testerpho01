**Apollo** - AI coding assistant that lives in your terminal.

## Installation

```bash
# Add this tap
brew tap i-luv-pho/testerpho01

# Install Apollo
brew install apollo

# Run the setup
apollo-setup

# Set your API key
export ANTHROPIC_API_KEY="your-key"

# Launch Apollo
apollo
```

## What is Apollo?

Apollo is an AI coding assistant built on the aichat + llm-functions stack. It can:

- Read and write files
- Execute shell commands
- Understand and modify codebases
- Save conversation sessions

## Requirements

- macOS or Linux
- Anthropic API key ([get one here](https://console.anthropic.com/))
